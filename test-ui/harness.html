The quadratic formula is: $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$ For inline math, if $ax^2 + bx + c = 0$, we can solve for $x$. Here's Euler's identity: $e^{i\pi} + 1 = 0$ And a display equation: $$\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$
